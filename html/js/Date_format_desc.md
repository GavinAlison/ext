## Date.format 
>日期格式化

>### 语法
```
Date.format(parameter);
```
#### 参数 parameter
```注意:/表示或者意思```
| 名 |  说明 |
|:--|:--|
| yyyy/YYYY | 年份-四位 |
| yy/YY   |  年份-二位 |
| MM |  月份-二位 |
| M  |  月份-一位 |
| w/W |  星期几 (0-6,0代表星期天)  |
| dd/DD | 当前日(1-31)- 二位  |  
| d/D | 当前日(1-31)--一位 |
| hh/HH | 	当前小时数(0-23 ) - 二位  | 
| h/H | 当前小时数(0-23 ) -一位 |
| mm/MM | 当前分钟数(0-59) -  二位  | 
| m/M | 当前分钟数(0-59) -  一位 |
| ss/SS |当前秒数(0-59) -  二位  | 
| s/S |当前秒数(0-59) -  一位  |
